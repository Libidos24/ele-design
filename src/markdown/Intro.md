# 介绍

Ele-design 是一个简单的前端 UI 框架，基于 vue3.0 开源 UI 组件库，使用 vite vue3.0
typescript 构建，如果你对框架的原生风格不满意，
不用担心类覆盖样式的命名也是可以的， ele-design 也支持移动显示。
